# Sea Waves Predictor

An AI-based intelligent agent system that predicts sea wave characteristics using environmental parameters.

## Features
- Predicts significant wave height (Hs)
- Estimates maximum and minimum wave heights
- Plots wave probability distribution
- Uses an intelligent agent approach

## Concept Used
- Artificial Intelligence (Intelligent Agents)
- Mathematical Modeling
- Probability Distribution
- Basic Machine Learning Correction

## Inputs Required
- Wind Speed (m/s)
- Wind Duration (hours)
- Fetch (km)
- Sea Depth (m)
- Tide Level (m)
- Current Speed (m/s)

## Outputs
- Significant Wave Height
- Maximum Wave Height
- Minimum Wave Height
- Probability Distribution Graph

## Technologies Used
- Python
- NumPy
- Matplotlib

## How to Run
1. Install dependencies:
   pip install numpy matplotlib
